### C的编译

```bash
gcc -c add.c -o add.o
```

这里是命令各部分的解释：

- `gcc`：调用GNU C编译器。
- `-c`：告诉`gcc`只编译源代码而不进行链接。这将生成一个目标文件（`.o`文件）。
- `add.c`：指定要编译的源文件。
- `-o add.o`：指定输出的目标文件名。如果不使用`-o`选项，`gcc`会默认生成与源文件同名的目标文件，但扩展名为`.o`，在这个例子中是`add.o`。

所以，如果你想编译`add.c`并希望输出文件名为`add.o`，你应该使用上面提供的命令。这样做会告诉`gcc`编译`add.c`文件，并将生成的目标文件保存为`add.o`。
